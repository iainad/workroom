# Ghostty bundled theme collection

These theme files are sourced from Ghostty's bundled theme collection (https://github.com/ghostty-org/ghostty), which itself vendors the iTerm2-Color-Schemes project (https://github.com/mbadolato/iTerm2-Color-Schemes), distributed under the MIT License (see LICENSE file in this directory).

## Curated themes

The following 12 theme families are included, with dark and light variants:

- Catppuccin Mocha, Catppuccin Latte
- TokyoNight Night, TokyoNight Day
- Gruvbox Dark, Gruvbox Light
- Nord, Nord Light
- One Half Dark, One Half Light
- GitHub Dark, GitHub Light Default
- Rose Pine, Rose Pine Dawn
- Everforest Dark Hard, Everforest Light Med
- Ayu Mirage, Ayu Light
- Kanagawa Wave, Kanagawa Lotus
- Kanso Zen, Kanso Pearl
- Monokai Pro, Monokai Pro Light

Additional in-repo themes ("Workroom", "Workroom Light") may be present; these are authored within this repository and not sourced from upstream.
